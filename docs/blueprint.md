# **App Name**: Abstergo Intel

## Core Features:

- AI-Powered Conversational Intelligence: Engage in natural language conversations to submit intelligence queries and receive AI-generated analysis, summaries, or insights.
- Flexible AI Provider Integration: Add, configure (API keys/URLs), and manage multiple large language model (LLM) providers (e.g., OpenAI, Anthropic, Gemini, Groq) and select specific models for each investigation.
- Investigation Lifecycle Management: Create, rename, duplicate, and delete distinct 'investigation' chats to organize different cases and maintain conversational context.
- Contextual Intelligence Agent: A generative AI tool that leverages current chat messages and attached files to reason about complex intelligence queries and provide relevant, structured analysis or data points.
- Rich Text & Media Message Display: View chat messages with full markdown rendering, including code blocks, lists, and formatted text, alongside visual cues for attached files.
- File Attachment & Management: Upload and attach various file types (images, documents) to individual messages and manage them within a chat's 'Vault' for future reference or AI processing (visual linking in MVP).

## Style Guidelines:

- Primary color: Vibrant indigo-violet blue (#594DFF), evoking futurism and analytical depth against the dark background.
- Background color: A very dark, desaturated blue-black (#18181F), providing a subtle yet distinct depth while maintaining a sleek, professional dark mode aesthetic.
- Accent color: A lighter, clear blue (#70C8FF) for highlights, interactive elements, and badges, providing high contrast and clarity.
- Body and headline font: 'Source Code Pro' (monospace sans-serif), offering a uniform, highly readable, and distinctly tech-oriented feel suitable for intelligence operations and code display alike.
